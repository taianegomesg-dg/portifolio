# 🚀 Meus Estudos de C e C++

Repositório organizado para o aprendizado de programação e estrutura de dados.

## 📂 Estrutura do Projeto
- **/c**: Exercícios em Linguagem C.
- **/cpp**: Exercícios em C++.
- **/bin**: Executáveis gerados automaticamente (organização profissional).

## 🛠️ Como Compilar e Rodar
O ambiente está configurado com **Tasks** personalizadas do VS Code.
- Basta abrir um arquivo `.c` ou `.cpp`.
- Pressionar `Ctrl + Shift + B`.
- O sistema compila e executa o código automaticamente no terminal.
